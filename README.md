# QR CODE SCANNER IN PYTHON

## Used technologies
    - OpenCV
    - Pzybar

## Getting Started
- Install OpenCV:
```
  <pip install opencv-contrib-python>
```
- Install Pzybar
```
     <pip install pyzbar>
```

## How to Run
```
   <python QR-Code-Scanner.py>
```

To close the camera window, press the keyboard Q key.
